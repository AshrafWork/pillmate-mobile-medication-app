# PillMate

## Overview

PillMate is a cross-platform mobile medication management application developed using Flutter and Dart.

The project was created as part of my final-year Software Engineering degree and was designed to address practical challenges encountered within pharmacy environments by helping users organise and manage their medications effectively.

## Features

- Medication management
- Medication scheduling
- Cross-platform mobile support
- User-friendly interface
- Pharmacy-inspired workflow design

## Technologies Used

- Dart
- Flutter
- Android Studio
- Visual Studio Code

## Project Background

This application was developed as a final-year Software Engineering project and achieved a First-Class grade (70%).
